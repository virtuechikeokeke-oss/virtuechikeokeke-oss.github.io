ISO 14971 Risk Assessment Generator

AI-powered web tool that generates pre-populated medical device risk assessment tables compliant with ISO 14971. Upload a device description or enter specs manually — the tool produces a fully formatted Excel file with 15–25 device-specific hazards, severity/probability ratings, mitigations, residual risk, and ISO clause references. Built for regulatory engineers and students who need a fast, accurate starting point for risk management documentation.

🔬 Research

EEG Brain-Computer Interface for Robotic Arm Control

3-year research project at URI's Neurorehabilitation Lab developing a real-time BCI system for individuals with motor impairments. Built a complete EEG signal processing pipeline — bandpass filtering, artifact rejection, spectral feature extraction — achieving ~85% motor intent classification accuracy. Fabricated custom tri-polar electrodes and programmed embedded microcontrollers (Arduino, C++) for closed-loop hardware control via OpenBCI Cyton.

Python MATLAB OpenBCI Cyton Arduino / C++ EEG Signal Processing

Adaptive Treadmill for Stroke Rehabilitation

Capstone design project building a sensor-driven rehabilitation treadmill with real-time adaptive control. Integrated heart rate and kinematic speed sensors to enable gait-responsive modulation, achieving closed-loop feedback latency under 150 ms. Designed adaptive feedback control algorithms to dynamically adjust speed and resistance based on individual patient physiology during rehab sessions.
